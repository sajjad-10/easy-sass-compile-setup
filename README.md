# easy-sass-compile-setup
This repo for crate easy sass compile setup. for your project. you can use online Editor. but this is real simple.

## Deployment

add node sass 

```bash
  npm i node-sass
```

for run type 

```bash
  npm run sass
```
## Usage/Examples
This generate rem size:

```css
h1{
    font-size: rem(100);  
}
```
